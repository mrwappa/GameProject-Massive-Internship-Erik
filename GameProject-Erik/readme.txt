This is the base repository for the LBS 10 week internship project.

Run code\GameProject\GameProject.sln

Set the correct working directory.
 - Right-click on the GameProject project in Visual Studio solution explorer.
 - Select Debugging in the left panel
 - Set working directory to be ..\..\data for all configurations.

Hit F5 to build and run.
 
Project has been tested in x64 Debug/Release.
